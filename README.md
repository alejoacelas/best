# best

My life-optimization workspace.

I'm a Toby Ord utilitarian. I take that to mean: there's a set of questions about
what's best — the best act, motive, trait, work, life — that we can keep asking
ourselves and each other, and that I want to hold my best tentative answers to and act
from, to make outcomes as good as possible. So this runs as a practice — **every folder
is a standing question, every leaf a tentative answer.**

Two moves I keep making, usable inside any folder:

- **`other/`** — when a category collapses into its current answers, ask what *other*
  ways there are to do this. Keeps the question bigger than my answers.
- **`later/`** — what's worth returning to, but not now. A question I'm choosing to
  defer, not a parking lot.

The **Questions** map below is those standing questions, generated from each folder's
`README.md`, so it always mirrors the live worktree. A few folders stay their own Git
repos (private, or standalone) and are listed under **Sub-repositories**.
